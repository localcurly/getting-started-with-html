Basics of the web

-The world wide web is made up of hyper text markup language documents, or html documents for short

-The web was invented in the 1990s and has an estimated 30 million pages.

-In order to function it has 3 major peices. (Your computer browser, the internet, and servers)

HTML & Markup

HTML is made up of:
-Text Content
- Markup
-References to other documents
-Links

Text content is rather self-explanitory, as for markup? -Not so much. Markup is made up of tags that tell one's computer what to do.

For example, < b > makes text bolded.
& < em > makes things italicized.

Computers are Stupid

Computers take the information they are given very literally. This makes them very sensitive to things like typos and other human errors.

Inline vs Block

HTML elements can be either inline or block, meaning they are either aligned with the text or, an "invisible box" is places around it.
